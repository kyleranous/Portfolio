Recipe Book

A Flask API for my Home Automation Setup

Project Description

Recipe Book is an API for storing and retreiving recipes. I am designing this API specifically to interface with my Home Assistant server, but as it is a basic RESTful API it could be integrated into just about anything. This project is under the MIT license and is free to use or modify by anybody. Be advised, there is no authentication on this API, anyone with access can add, modify, or delete recipes.

Github

The front end for my Recipe Book will be handled with Home Assistant Cards. My family calendar and shopping list are both linked to my Home Assistant server so meals can be scheduled on the calendar and ingredients can be added to the shopping list as needed. When a recipe is added to the calendar, home assistant can use the stored prep and cook times to back plan on when meal preperation needs to begin. A walk through on my Home Assistant implementation can be found here.

Planned Integrations

  • Recipe Search - Search by Name or ingredients you want to use, or specifically ingredients you want to exclude if you have someone with a food sensitivity.
  • Web Scraper or Browser Plugin to add recipes found online to the Recipe Book
  • Email connectivity to send formatted recipes to people
Currently I am only intending to use this on my internal network for my family. However It could be expanded into a social feature where people could subscribe to other peoples meal prep. A person could create a meal plan and the recipes and schedule could automatically be sent to subscribers. This could also be a powerful method of teaching people how to do proper meal planning.
